- Content box: where the padding/border is added on top of the configuration of the element.
- Border box: where the padding/border is added within the div thus the div measurements don't increase in any way.


- Margin collapse: only applies to vertical spacing. That is the block element with the bigger vertical margin prevails while the smaller one collapses. But in horizontal layouts the margins add up
	- Margin collapse is negated by using flex, grid, position layouts or 
	 display: flow-root
