# Normalisation
---
1NF 
1) Rows must be unique
2) Every table should have a primary key

2NF
1) Each non-key attribute must depend on the entire primary key

3NF
1) Every non-key attribute in a table should depend on the key, the whole key and only on the key

4NF

5NF
1) The table cannot be describable as the logical result of joining some other tables
